## 🚀 快速开始

1.  **创建一个 Gatsby 站点。**

    使用 Gatsby CLI 创建一个新的站点，指定最小的 TypeScript 初始模板。

    ```shell
    # 使用最小 TypeScript 初始模板创建一个新的 Gatsby 站点
    npm init gatsby -- -ts
    ```

2.  **开始开发。**

    进入新站点的目录并启动它。

    ```shell
    cd my-gatsby-site/
    npm run develop
    ```

3.  **打开代码并开始自定义！**

    你的站点现在运行在 http://localhost:8000!

    编辑 `src/pages/index.tsx`，实时查看你的站点更新！
